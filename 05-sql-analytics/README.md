## 🏗️ Схема БД (3NF)

```
countries ──► customers ──► orders ──► order_items
                ▲              ▲
                └──────────────┘
                products ◄─────┘
```

- `countries` — справочник (35 стран)
- `customers` — 2,708 клиентов + агрегаты
- `products` — 3,110 товаров
- `orders` — 7,358 заказов
- `order_items` — 144K строк

## 🎯 Ключевые SQL-концепции

- **CTE (WITH)** — временные таблицы для читаемости
- **Оконные функции** — NTILE, LAG, LEAD, RANK, ROW_NUMBER
- **Frame specification** — ROWS BETWEEN для скользящих средних
- **JOIN 4 таблиц** — многотабличные запросы
- **GROUP BY + HAVING** — фильтрация агрегатов

## 💡 Инсайты

- **Champions** (25.9%) — самый большой сегмент, отличная база
- **Lost** (25.4%) — огромный потенциал для win-back
- **UK** = 94% выручки — концентрация риска
