## 🎯 Ключевые концепции

- **NTILE(n)** — делит на n равных групп (квинтили, квартили)
- **LAG(col, N)** — значение из строки на N позиций назад
- **LEAD(col, N)** — значение из строки на N позиций вперёд
- **ROWS BETWEEN** — задаёт фрейм окна (скользящее среднее, кумулятивные суммы)
- **RANK()** — ранг с пропусками при совпадениях
- **DENSE_RANK()** — ранг без пропусков
- **ROW_NUMBER()** — просто номер строки
- **CUME_DIST()** — накопленная доля (0..1)
- **PERCENT_RANK()** — ранг в процентах
- **FIRST_VALUE / LAST_VALUE** — первое/последнее значение в окне
