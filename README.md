## 🎯 Ключевая идея: Augmented Analyst

### Классический подход
Аналитик в одиночку: SQL + Excel/Python + BI + выводы = **4–6 часов на задачу**.

### Augmented Analyst (мы)
- **Евгений** (Human): стратегия, вопросы, бизнес-контекст, валидация
- **Jack** (Hermes Agent): генерация данных, SQL-шаблоны, статистика, визуализация, документация
- **Subagents**: параллельная обработка (один считает метрики, другой проверяет значимость, третий пишет выводы)
- **Second Brain**: Obsidian Vault + SQLite — всё сохраняется и переиспользуется

**Результат:** задача решается за **30–40 минут** с более высоким качеством.

## 📊 Результаты (примерные)

### Воронка продаж
- Конверсия: просмотр → корзина → покупка = 100% → 40% → 25%
- Лучший канал: `referral` (11.5% конверсии)
- Проблема: `mobile` — только 9.2% конверсии (проверить UX)

### A/B тест
- Вариант B: +24.6% конверсии (p = 0.000012)
- ARPU прирост: +28.0%
- Рекомендация: раскатать B на 100% трафика

### RFM + Когортный анализ (реальные данные)
- **Champions** (9.8% клиентов) приносят 32% выручки
- **Lost** (40%) — резерв для реактивации, $650K выручки
- **Retention** падает к 6 мес. до ~20%
- **EU рынок** недооценён: Германия, Франция, Нидерланды
